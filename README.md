# CodeLeveling

CodeLeveling is a desktop learning platform that teaches C++ concepts through
quests, lessons, and quizzes.

## Features
- Quest-based progression
- Lessons + quizzes per quest
- XP & leveling system
- SQLite local database
- Qt + QML UI

## Tech Stack
- C++ (Qt 6)
- QML / QtQuick
- SQLite
