# CodeLeveling

CodeLeveling is a desktop learning platform that teaches C++ concepts through
quests, lessons, and quizzes. This is a remake of my original CodeLeveling project.
This version is a fresh restart with a clearer vision, improved structure, and a refined learning experience.

## Features
- Quest-based progression
- Lessons + quizzes per quest
- XP & leveling system
- SQLite local database
- Qt + QML UI

## Tech Stack
- C++ (Qt 6)
- QML / QtQuick
- SQLite
